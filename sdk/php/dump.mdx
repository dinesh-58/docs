---
title: PHP SDK "dump"
description: It prints the Rowset on standard output with SQLiteCloudRowset.dump()
---

```js
SQLiteCloudRowset.dump()
```

Print the Rowset on standard output.

### Example
```js

$sqlitecloud = new SQLiteCloud();
$sqlitecloud->username = 'admin';
$sqlitecloud->password = 'admin';

try {
	if ($sqlitecloud->connect('mynode.sqlite.cloud', 8860) == false) {
		$msg = $sqlitecloud->errmsg;
		$sqlitecloud = NULL;
		return $msg;
	}
  } catch (Exception $e) {
		return $e->getMessage();
  }

$result = $sqlitecloud->execute('LIST INFO');
$result->dump();

$sqlitecloud->disconnect();
```
