## Syntax

GET USER

## Privileges

```
NONE
```

## Description

The GET USER command returns the username of the currency-connected user.

## Return

A [String](https://github.com/sqlitecloud/sdk/blob/master/PROTOCOL.md) set to the current username.

## Example

```bash
> GET USER
admin
```
